WHAT IS WEBSOCKET?WebSocket is a protocol providing full-duplex communications channels over a single TCP connection. The WebSocket protocol was standardized by the IETF as RFC 6455 in 2011, and the WebSocket API in Web IDL is being standardized by the W3C.HOW TO CONNECT?Bitstamp uses Pusher for real time websocket streaming. Please refer to the Pusher documentation and libraries to connect to our websocket stream. You can also find an example for each stream below.LIVE TICKERCHANNEL: live_trades, EVENT: trade, PUSHER KEY: de504dc5763aeef9ff52Streams JSON dictionary:
id - trade unique ID
amount - trade amount
price - trade price
JavaScript example: live_trades.htmlLIVE ORDER BOOKCHANNEL: order_book, EVENT: data, PUSHER KEY: de504dc5763aeef9ff52Streams two JSON lists:
bids - list of top 20 bids
asks - list of top 20 asks
JavaScript example: live_order_book.html
